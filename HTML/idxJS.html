Practica 1

Contador con Botones

Instrucción: Se creará una página web con tres botones: aumentar, disminuir y reiniciar el valor de un contador,
con el objetivo de practicar la manipulación de eventos y la actualización dinámica de contenido
mediante JavaScript.

Practica 2

Mayúsculas y Minúsculas

Instrucción: Se solicita diseñar una página web con un párrafo de texto editable y dos botones, uno para convertir
todo el texto a mayúsculas y otro a minúsculas, usando funciones en JavaScript para cambiar dinámicamente
el contenido del párrafo según el botón presionado.

Practica 3

Operaciones Aritmeticas

Instrucción: Se debe crear una mini calculadora web que permita ingresar dos números y seleccionar una operación
básica (+, –, *, /). Al presionar un botón, el símbolo cambia y se muestra el resultado directamente en
pantalla usando JavaScript.

Practica 4

Calculo del IMC

Instrucción: Se requiere una página que calcule el IMC al ingresar peso y estatura del usuario. Usando JavaScript
y una estructura switch, se mostrará la categoría correspondiente del resultado: bajo peso, normal,
sobrepeso o algún grado de obesidad.
